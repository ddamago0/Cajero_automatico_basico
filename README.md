# Cajero_automatico_basico
Simulador de Cajero Automático 

Programa en Python para gestionar transacciones bancarias básicas a través de la consola.

Funcionalidades:

    Consulta de saldo: Revisa fondos actuales (1000 iniciales).

    Depósitos y Retiros: Actualización automática del saldo.

    Validaciones: Control de montos negativos y fondos insuficientes.

    Límite de operaciones: El usuario define cuántas transacciones realizar.
